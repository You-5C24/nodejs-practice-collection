# 文件管理工具 (File Manager CLI)

一个基于交互式命令行界面的文件管理工具，用于学习 Node.js 的文件系统操作。

## 📖 学习目标

- **fs 模块：** 文件系统操作（读取、写入、删除、重命名）
- **path 模块：** 路径处理和解析
- **inquirer 库：** 交互式命令行界面开发
- **异步操作：** Promise、async/await、回调函数
- **流处理：** 处理大文件的读写

## 🎯 功能特性

### 1. 文件搜索
- 按文件名搜索
- 按文件类型过滤
- 递归搜索子目录

### 2. 批量重命名
- 添加前缀/后缀
- 替换文件名部分
- 批量修改文件扩展名

### 3. 文件统计
- 统计文件数量
- 计算文件大小
- 分析文件类型分布

## 🚀 安装和使用

```bash
# 安装依赖
pnpm install

# 运行程序（启动交互式界面）
pnpm start

# 或直接运行
node src/index.js
```

### 使用流程

1. 运行程序后，会显示主菜单供您选择操作
2. 选择 **🔍 搜索文件** 或 **✏️ 批量重命名**
3. 根据提示输入相应的参数
4. 查看操作结果
5. 选择是否继续使用或退出

### 功能说明

#### 搜索文件
- 输入目录路径（默认当前目录）
- 输入搜索关键词（可选）
- 输入文件扩展名过滤（可选，如 .js .txt）
- 程序会递归搜索并显示匹配的文件

#### 批量重命名
- 输入目标目录路径
- 选择重命名模式：
  - **添加前缀**：在文件名前添加指定文本
  - **添加后缀**：在文件名后（扩展名前）添加指定文本
  - **替换文本**：替换文件名中的指定文本
  - **数字编号**：使用数字序列重命名文件
- 可选择仅处理特定扩展名的文件
- 支持预览模式，先查看效果再决定是否实际修改

## 💡 实现要点

1. 使用 `fs.promises` API 处理异步文件操作
2. 使用 `path.join()` 和 `path.resolve()` 处理路径
3. 使用递归函数遍历目录树
4. 错误处理和边界情况考虑
