# 文件管理工具 (File Manager CLI)

一个基于命令行的文件管理工具，用于学习 Node.js 的文件系统操作。

## 📖 学习目标

- **fs 模块：** 文件系统操作（读取、写入、删除、重命名）
- **path 模块：** 路径处理和解析
- **命令行参数：** process.argv 的使用
- **异步操作：** Promise、async/await、回调函数
- **流处理：** 处理大文件的读写

## 🎯 功能特性

### 1. 文件搜索
- 按文件名搜索
- 按文件类型过滤
- 递归搜索子目录

### 2. 批量重命名
- 添加前缀/后缀
- 替换文件名部分
- 批量修改文件扩展名

### 3. 文件统计
- 统计文件数量
- 计算文件大小
- 分析文件类型分布

## 🚀 安装和使用

```bash
# 安装依赖
npm install

# 运行程序
npm start

# 或直接运行
node src/index.js
```

## 💡 实现要点

1. 使用 `fs.promises` API 处理异步文件操作
2. 使用 `path.join()` 和 `path.resolve()` 处理路径
3. 使用递归函数遍历目录树
4. 错误处理和边界情况考虑

## 📚 相关知识点

- Node.js 文件系统 API
- 异步编程模式
- 命令行工具开发
