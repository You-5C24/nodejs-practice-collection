# 网页爬虫 (Web Scraper)

一个简单的网页爬虫工具，用于学习 HTTP 请求和 HTML 解析。

## 📖 学习目标

- **HTTP 请求：** axios/node-fetch 的使用
- **HTML 解析：** cheerio/jsdom 的使用
- **数据提取：** CSS 选择器和 DOM 操作
- **流处理：** 处理大量数据
- **数据存储：** 将数据保存到文件或数据库
- **异步控制：** 并发请求和速率限制

## 🎯 功能特性

### 1. 网页抓取
- 发送 HTTP 请求
- 获取网页内容
- 处理重定向和错误

### 2. 数据提取
- 使用 CSS 选择器定位元素
- 提取文本和属性
- 清洗和格式化数据

### 3. 数据存储
- 保存到 JSON 文件
- 保存到 CSV 文件
- 存储到数据库

## 🚀 安装和使用

```bash
# 安装依赖
npm install

# 运行爬虫
npm start

# 指定目标 URL
node src/index.js <url>
```

## 💡 实现要点

1. 使用 axios 发送 HTTP 请求
2. 使用 cheerio 解析 HTML
3. 实现数据提取逻辑
4. 处理异步请求和错误
5. 实现速率限制避免封禁
6. 尊重 robots.txt 规则

## 📚 相关知识点

- HTTP 协议基础
- HTML 结构和 CSS 选择器
- 异步编程和 Promise
- 数据清洗和格式化
- 爬虫礼仪和法律问题

## ⚠️ 重要提示

- 遵守网站的 robots.txt 规则
- 控制请求频率，避免对服务器造成压力
- 尊重网站的服务条款
- 不要爬取敏感或受版权保护的内容
- 仅用于学习目的
