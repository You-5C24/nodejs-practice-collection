# 实时聊天应用 (Real-time Chat)

一个基于 WebSocket 的实时聊天应用，用于学习 Socket.io 和实时通信技术。

## 📖 学习目标

- **Socket.io：** 实时双向通信库的使用
- **WebSocket：** WebSocket 协议理解
- **事件驱动：** 事件的发送和监听
- **实时通信：** 客户端与服务器的实时交互
- **广播机制：** 消息的广播和私聊

## 🎯 功能特性

### 1. 多用户聊天室
- 用户加入/离开通知
- 实时消息发送
- 消息历史记录

### 2. 用户管理
- 用户昵称设置
- 在线用户列表
- 用户状态显示

### 3. 消息功能
- 公共消息广播
- 消息时间戳
- 系统通知消息

## 🚀 安装和使用

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问应用
# 打开浏览器访问 http://localhost:3000
```

## 💡 实现要点

1. 使用 Express 提供静态文件服务
2. 集成 Socket.io 实现 WebSocket 通信
3. 监听连接/断开事件
4. 实现消息广播机制
5. 管理在线用户列表

## 📚 相关知识点

- Socket.io 基础
- WebSocket 协议
- 事件驱动编程
- 客户端-服务器通信
- 实时数据同步
