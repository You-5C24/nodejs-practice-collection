# Node.js 实践项目集合

这是一个系统化学习和巩固 Node.js 知识的项目集合。每个子项目都专注于不同的 Node.js 核心概念和实际应用场景。

## 📚 项目列表

### 1. [文件管理工具](./01-file-manager-cli/)
**学习目标：** fs模块、path模块、命令行参数、异步操作
- 文件搜索和过滤
- 批量重命名操作
- 文件统计分析

### 2. [RESTful API服务器](./02-simple-rest-api/)
**学习目标：** Express框架、中间件、路由、HTTP方法、错误处理
- CRUD操作实现
- 用户管理功能
- 数据验证和错误处理

### 3. [实时聊天应用](./03-real-time-chat/)
**学习目标：** Socket.io、WebSocket、事件驱动、实时通信
- 多用户聊天室
- 实时消息广播
- 用户在线状态管理

### 4. [任务管理API](./04-task-manager-api/)
**学习目标：** 数据库集成、ORM/ODM、身份验证、JWT
- 任务增删改查
- 用户认证和授权
- Token管理

### 5. [网页爬虫](./05-web-scraper/)
**学习目标：** HTTP请求、HTML解析、数据提取、流处理
- 网站内容抓取
- 数据清洗和结构化
- 数据存储

### 6. [邮件发送服务](./06-email-service/)
**学习目标：** Nodemailer、模板引擎、定时任务
- 邮件发送功能
- 邮件模板管理
- 批量邮件发送

### 7. [URL短链接服务](./07-url-shortener/)
**学习目标：** 数据库设计、哈希算法、重定向、缓存
- 短链接生成
- URL重定向
- 访问统计分析

### 8. [图片处理服务](./08-image-processor/)
**学习目标：** Stream、Buffer、文件上传、第三方库集成
- 图片压缩
- 格式转换
- 缩略图生成

### 9. [日志分析工具](./09-log-analyzer/)
**学习目标：** Stream处理、正则表达式、数据分析
- 日志文件解析
- 统计分析
- 报告生成

### 10. [微服务通信演示](./10-microservices-demo/)
**学习目标：** 微服务架构、进程间通信、负载均衡
- 多服务协作
- 消息队列
- 服务发现

## 🚀 快速开始

每个项目都是独立的，可以单独学习和运行：

```bash
# 进入任意项目目录
cd 01-file-manager-cli

# 安装依赖
npm install

# 运行项目
npm start
```
